# VScode-markdown-style
- [원본 파일](https://github.com/nicolashery/markdownpad-github)에서 수정
- 글씨 font 변경 : D2Coding
- 글씨 size 변경 : 16


## 사용법
1. VSCode **settings.json**
2. 아래 코드 입력
```json
"markdown.styles": [
        "https://cdn.jsdelivr.net/gh/Daikoku1/VScode-markdown-style/Markdown_style.min.css"
]
```
